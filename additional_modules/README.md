# Additional Modules

This directory keeps the repository's secondary coursework modules available without mixing them into the main project narrative.

These modules are not part of the primary computational neuroscience study documented in the top-level `README.md`:

- `05_associative_networks`
- `06_hopfield_reconstruction`
- `07_hopfield_extensions`
- `08_backprop_single_hidden_layer`
- `09_mnist_classification`
- `10_competitive_networks`

They may still be useful as supporting examples or as the basis for future standalone projects.
