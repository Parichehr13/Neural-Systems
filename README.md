# Neural Systems

- `01_integrate_and_fire`: Integrate-and-fire neuron models (constant and variable threshold) and current-frequency behavior.
- `02_adaptive_neuron`: Integrate-and-fire model with adaptation conductance.
- `03_coupled_neurons`: Two coupled integrate-and-fire neurons with synaptic interaction and synchrony analysis.
- `04_neural_mass_model`: Neural mass model simulation (EEG-like signal and power spectral density).
- `05_associative_networks`: Hetero-associative neural networks with noisy-pattern recognition and reconstruction.
- `06_hopfield_reconstruction`: Hopfield network for image storage and recovery from corrupted inputs.
- `07_hopfield_extensions`: Hopfield model extensions (dilution, low M/N ratio, sparse coding).
- `08_supervised_learning`: Supervised learning of chemoreceptor response with a small neural network.
- `09_mnist_classification`: Deep learning for handwritten digit classification (MNIST-based workflow).
- `10_competitive_networks`: Competitive neural network with lateral inhibition for contrast/resolution effects.
